# Практична робота №1: Основи HTML

## Опис проєкту
Це навчальний міні-сайт із трьох сторінок, створений для вивчення базових тегів HTML5, структури документа, форм та медіа-елементів.

## Структура проєкту
- `index.html` — головна сторінка (заголовки, списки, таблиця).
- `pages/about.html` — сторінка "Про мене" з вбудованим відео та аудіо.
- `pages/contact.html` — сторінка з формою зворотного зв'язку.
- `assets/` — папка для мультимедіа та зображень.

## Як запустити
1. Клонувати репозиторій.
2. Відкрити файл `index.html` у будь-якому сучасному браузері або скористатися розширенням Live Server у VS Code.
